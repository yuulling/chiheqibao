# 汽保工具网站 — 项目记忆

## 技术栈
- Next.js 14 (App Router) + TypeScript + Tailwind CSS
- Prisma ORM + SQLite (本地开发)
- iron-session 认证
- 部署到 Vercel (免费)

## 已实现功能
- 前台：首页、产品列表(分类筛选/搜索/分页)、产品详情(图片轮播/规格表)、关于我们、联系我们
- 后台：登录、仪表盘、产品CRUD(新增/编辑/删除)、分类CRUD(弹窗管理)、网站设置
- 种子数据：管理员(admin/admin123)、6个分类、6个示例产品

## 重要命令
- `npm run dev` — 启动开发服务器
- `npm run build` — 构建生产版本
- `npm run db:seed` — 重新初始化种子数据
